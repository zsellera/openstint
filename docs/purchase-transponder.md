# Transponders

This project is accompanied by an open-source [transponder project](https://github.com/zsellera/openstint-transponder) as well.

You can manufacture them at [JLCPCB](https://jlcpcb.com), a popular Chinese fabhouse. 5 panels (40 pcs tranponders) cost me $200, including taxes and shipping to EU (2026 May). The fabhouse produces the PCB and assembles with components - you have to program the ATTiny, solder the wires and deal with the mechanical protection.

![OpenStint Transponders](https://openstint.org/openstint-transponder.png)
